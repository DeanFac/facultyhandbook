# Faculty academic profile 

The [Faculty Academic Profile](https://fap.iitm.ac.in) is for faculty to update academic data and achievements annually. This data is used by the Insitute to track achievements for high level positions.
The reports created from the submitted FAP data can be viewed at [https://workflowreports.iitm.ac.in/](https://workflowreports.iitm.ac.in/) :arrow_right: HumanResources :arrow_right: FAP
